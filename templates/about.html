Welcome to the BULBS web app!

This web app was developed as part of my CS50x final project! The idea for this app was inspired by the "Bulbs" homework assignment from CS50x. This web app is designed to help you transform text messages into captivating patterns of bulbs using binary code, providing an interactive learning experience along the way.

How does it work?

  1. Input your text message: Simply enter the desired text into our user-friendly interface.
  2. Conversion process: Behind the scenes, the Flask backend takes each character of your message and converts it into its corresponding unicode value.
  3. Binary encoding: The unicode values are then transformed into an 8-bit binary representation, creating a unique binary message.
  4. Visual representation: Each 0 and 1 in the binary message corresponds to the state of an individual bulb. As the binary message unfolds, the bulbs light up or remain dim, creating an intriguing pattern.

Decode Binary Messages

In addition to encoding text messages into binary patterns, our web app allows you to decode received binary messages back into plain text. If you have received a mesmerizing binary pattern, simply navigate to the Decode page and input the binary message. Our app will convert it back into meaningful text for you to decipher.

The Vision

My goal for this app is to make learning about binary code fun and accessible. I hope that visual representations and interactive experiences can enhance the learning process, enabling you to grasp complex concepts with ease. Whether you're a student, a technology enthusiast, or simply curious about binary, our web app is here to inspire and educate.
